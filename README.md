## Что сюда заносить
- маршруты и планы поездок;
- расходы: топливо, жильё, еда, обмен валют;
- заметки по дорогам, границам, связи;
- полезные адреса, контакты, цены;
- списки вещей и документов;
- впечатления и рекомендации по странам.

## Структура
```
travel-notes/
├─ routes/          — маршруты поездок
├─ countries/       — страновые заметки (Грузия, Турция, Кипр...)
├─ budgets/         — расходы и бюджеты
├─ packing-lists/   — списки вещей
└─ notes/           — практические заметки
```
